LDAP Maven Plugin

This plugin provides the goals to:

  • Load records into or dump records from an LDAP directory. Records can be dumped to LDIF or DSML formats.
  • Launch and shutdown one of the following embedded directory server within the Maven build life-cycle:
    • ApacheDS
    • OpenDJ
    • UnboundID

Goals Overview

  • ldap:load Load directory entries from DSML or LDIF format.
  • ldap:dump Dump directory entries to DSML LDIF format.
  • ldap:run Run an embedded ApacheDS server.
  • ldap:stop Stop an embedded ApacheDS server.
